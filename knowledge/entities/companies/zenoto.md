# Zenoto

- **Description:** Digital repository for persistent preservation and DOI minting, operated by CERN
- **Status:** Active repository service
- **Maintained by:** [[CERN]]

## Features

- **DOI Generation:** Free DOI (Digital Object Identifier) minting (~$1 per DOI)
- **Persistent Storage:** Long-term preservation of digital assets
- **Two-Level DOI Structure:**
  - Collection-level DOI (umbrella)
  - Version-specific DOIs
- **GitHub Integration:** Plugin for automatic publishing on GitHub releases
- **Metadata Management:** Tracks author, document metadata, citation metrics
- **Public Access:** Linked to GitHub with fallback preservation
