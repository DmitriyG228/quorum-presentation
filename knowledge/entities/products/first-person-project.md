# First Person Project

- **Description:** Initiative using decentralized trust credentials for Linux kernel and system verification
- **Status:** Active development and demonstration phase
- **Maintained by:** [[Linux Foundation]], partnering with [[Trust Over IP]]
- **Use Case:** Human agency in AI and kernel security

## Key Demonstration

- **Human Agency Summit (H2) Demo:** Feb 23, 2026
  - Demonstrating kernel credentials using decentralized trust graph
  - Presenters: [[Drummond Reed]], [[Jim Zen]]

- **Decentralized Trust Graph:** Credential exchange infrastructure
- **Kernel Verification:** Credential-based verification for Linux kernel
- **Agentic AI Integration:** Credentials for AI agent verification
